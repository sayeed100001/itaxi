const response = await fetch('/api/admin/stats');

### 2️⃣ **AdminDriversPage** (مدیریت رانندگان)
**مسیر:** `/pages/Admin/AdminDriversPage.tsx`

✅ **ویژگیها:**
- لیست کامل رانندگان از دیتابیس
- جستجو بر اساس نام، پلاک، تلفن
- فیلتر بر اساس وضعیت (available, busy, offline, suspended)
- ویرایش اطلاعات راننده (نام، تلفن، خودرو، پلاک، نوع سرویس)
- تعلیق/فعالسازی راننده
- نمایش درآمد واقعی راننده
- نمایش اعتبار خریداری شده

✅ **عملیات CRUD:**
```typescript
// خواندن
const drivers = useAppStore((state) => state.drivers);

// بروزرسانی
await updateDriverInDB(driverId, { name, phone, vehicle, ... });

// تغییر وضعیت
await fetch(`/api/drivers/${id}/status`, {
    method: 'PUT',
    body: JSON.stringify({ status: 'suspended' })
});
```

✅ **دادههای داینامیک:**
- Total Fleet: از دیتابیس
- Online Drivers: محاسبه لحظهای
- Average Rating: میانگین واقعی
- Suspended: شمارش از دیتابیس

### 6️⃣ **AdminAnalyticsPage** (آمار و تحلیل)
**مسیر:** `/pages/Admin/AdminAnalyticsPage.tsx`

✅ **ویژگیها:**
- نمودار تقاضا و درآمد (24 ساعت گذشته - داینامیک)
- توزیع ناوگان بر اساس نوع سرویس
- KPI های کلیدی: رانندگان فعال، درآمد کل، سفرها، مناطق فعال
- نقشه حرارتی تقاضا
- شاخصهای کیفیت: رضایت مشتری (CSAT)، دقت زمانی

✅ **دادههای واقعی:**
```typescript
// نمودار ساعتی - از دیتابیس
SELECT 
    strftime('%H:00', created_at) as time,
    COUNT(*) as rides,
    SUM(fare) as revenue
FROM rides
WHERE created_at > datetime('now', '-24 hours')
GROUP BY strftime('%H', created_at)

// CSAT - میانگین واقعی امتیازات
SELECT AVG(rider_rating) FROM rides WHERE rider_rating IS NOT NULL
```

## 🔗 اتصال به دیتابیس SQLite

### API Endpoints اضافه شده:

```typescript
// مدیریت سفرها
GET  /api/admin/rides              // لیست تمام سفرها
GET  /api/rides/user/:userId       // سفرهای یک کاربر

// مدیریت کاربران
GET  /api/admin/users              // لیست تمام کاربران
PUT  /api/admin/users/:id/status   // تعلیق/فعالسازی

// مدیریت رانندگان
GET  /api/drivers                  // لیست رانندگان
PUT  /api/drivers/:id              // ویرایش راننده
PUT  /api/drivers/:id/status       // تغییر وضعیت

// آمار و تحلیل
GET  /api/admin/stats              // آمار کامل سیستم

// مالی
GET  /api/admin/financials         // درخواستهای مالی
POST /api/admin/financials/approve // تایید/رد

// تنظیمات
GET  /api/admin/settings           // دریافت تنظیمات
PUT  /api/admin/settings           // ذخیره تنظیمات
```
