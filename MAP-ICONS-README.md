# سیستم آیکون های نقشه iTaxi

این سیستم آیکون های واقعی و هوشمند برای نمایش انواع مختلف تاکسی روی نقشه فراهم می کند.

## ویژگی ها

### 🚗 انواع تاکسی

1. **اقتصادی (Eco)** - `eco.svg`
   - تاکسی های ساده و مقرون به صرفه
   - کرایه پایه: 30,000 افغانی
   - هر کیلومتر: 5,000 افغانی
   - اندازه آیکون: 40x40 پیکسل

2. **پلاس (Plus)** - `plus.svg`
   - تاکسی های خاص با امکانات بیشتر
   - کرایه پایه: 45,000 افغانی
   - هر کیلومتر: 7,000 افغانی
   - اندازه آیکون: 45x45 پیکسل

3. **لوکس (Luxury)** - `lux.svg`
   - تاکسی های بزرگ و لوکس
   - کرایه پایه: 70,000 افغانی
   - هر کیلومتر: 10,000 افغانی
   - اندازه آیکون: 50x50 پیکسل

4. **پریمیوم (Premium)** - `premium.svg`
   - تاکسی های بزرگتر با سرویس VIP
   - کرایه پایه: 100,000 افغانی
   - هر کیلومتر: 15,000 افغانی
   - اندازه آیکون: 55x55 پیکسل

### 🎯 تشخیص خودکار نوع تاکسی

سیستم بر اساس رتبه و تعداد سفرهای راننده، نوع تاکسی را خودکار تشخیص می دهد:

- **پریمیوم**: رتبه ≥ 4.8 و سفرها ≥ 1000
- **لوکس**: رتبه ≥ 4.5 و سفرها ≥ 500  
- **پلاس**: رتبه ≥ 4.0 و سفرها ≥ 100
- **اقتصادی**: سایر موارد

### 🎨 ویژگی های بصری

- **انیمیشن پالس**: نشان دهنده فعالیت تاکسی
- **نشانگر وضعیت**: نقطه سبز برای آنلاین، قرمز برای مشغول
- **چرخش آیکون**: بر اساس جهت حرکت (bearing)
- **سایه و افکت**: برای بهتر دیده شدن روی نقشه

### 📍 اطلاعات پاپ آپ

هر آیکون تاکسی شامل اطلاعات کاملی در پاپ آپ:

- نام راننده
- نوع و مدل خودرو
- رتبه و تعداد سفرها
- شماره پلاک
- وضعیت فعلی (آنلاین/مشغول/آفلاین)
- کرایه پایه
- ویژگی های خودرو

## استفاده

### در کامپوننت نقشه

```typescript
import { determineTaxiType } from '../../services/taxiTypes';

// تعیین نوع تاکسی
const taxiType = determineTaxiType(driver);
// نمایش آیکون های تاکسی (SVG) داخل `RealMap` انجام می شود.
```

### انتخاب نوع تاکسی

```typescript
import { TaxiTypeSelector } from '../../components/Ride/TaxiTypeSelector';

<TaxiTypeSelector
    selectedType={selectedTaxiType?.id || 'eco'}
    onTypeSelect={setSelectedTaxiType}
    distance={5}
    duration={15}
    surgeMultiplier={1.2}
/>
```

### محاسبه کرایه

```typescript
import { calculateFare } from '../../services/taxiTypes';

const fare = calculateFare(taxiType, distance, duration, surgeMultiplier);
```

## پیکربندی

### تغییر تصاویر

برای تغییر آیکون های تاکسی، فایل های SVG جدید را در پوشه `public/img/map-icons/` قرار دهید و مسیرها را در `TAXI_TYPES` بروزرسانی کنید.

### تغییر قیمت ها

قیمت ها در `TAXI_TYPES` قابل تنظیم هستند:

```typescript
export const TAXI_TYPES = {
    eco: {
        baseFare: 30000,    // کرایه پایه
        perKmRate: 5000,    // هر کیلومتر
        // ...
    }
};
```

### تغییر معیارهای تشخیص

معیارهای تشخیص خودکار در تابع `determineTaxiType` قابل تنظیم هستند.

## مزایا

✅ **واقعی**: استفاده از تصاویر واقعی تاکسی  
✅ **هوشمند**: تشخیص خودکار نوع بر اساس عملکرد راننده  
✅ **یکپارچه**: ادغام کامل با سیستم نقشه و پرداخت  
✅ **قابل تنظیم**: امکان تغییر آسان تصاویر و قیمت ها  
✅ **بهینه**: عملکرد بالا با انیمیشن های نرم  
✅ **چندزبانه**: پشتیبانی از فارسی و انگلیسی  

## نکات فنی

- آیکون ها با `L.divIcon` ایجاد می شوند
- انیمیشن ها با CSS3 پیاده سازی شده اند
- موقعیت GPS با انیمیشن نرم بروزرسانی می شود
- پاپ آپ ها RTL و فارسی هستند
- سیستم با تمام ارائه دهندگان نقشه سازگار است
